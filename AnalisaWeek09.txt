Aplikasi yang dibuat hari ini bertujuan untuk menyimpan dan mengambil data dari sqlite dan menggunakan Room, yang menggunakan DAO untuk menjalankan query ke database. Room juga digunakan agar data yang diambil dari sqlite dapat diterima dalam bentuk java data objek. Karena sqlite disimpan secara lokal, data dapat diterima tanpa koneksi internet dan dapat diakses walaupun aplikasi sudah ditutup. Aplikasi berjalan sesuai harapan.
