Projek ini dilakukan dengan memanfaatkan fitur google maps. Karena google maps merupakan service external dari google, kita perlu memberikan key api agar google maps terhubung dengan aplikasi yang dibuat.
Terdapat 3 class yang dibuat. Yang pertama adalah MarkerDanShape. Class ini digunakan untuk menyimpan koordinat dan tipe marker apa yang diberikan (titik, circle, garis, polygon). titik disimpan dalam arraylist karena suatu class bisa mempunyai multiple titik (polygon&line).  MainActivity digunakan untuk menginisialisasi marker yang sudah diberikan. Marker diinisialisasi di UMN dengan circle yang digunakan untuk menunjukkan area 500 m dari UMN, polygon yang digunakan untuk menunjukkan area kampus, dan line untuk menunjukkan rute dari bethsaida ke umn. MainActivity juga digunakan untuk mengambil semua objek dari class MarkerDanShape. Class terakhir adalah TambahObject. Class ini digunakan untuk menambahkan marker baru dengan menggunakan radiogroup dan pilihan yang akan dicheck dengan menggunakan switch. Semua data kemudian akan diadd ke class MarkerDanShape. MainActivity kemudian akan menampilkan initialized point sebelumnya dan juga poin yang sudah ditaruh. Karena tidak menyimpan state dengan menggunakan preference ataupun internal storage, semua point yang dibuat akan hilang jika aplikasi ditutup ataupun dirotate.
